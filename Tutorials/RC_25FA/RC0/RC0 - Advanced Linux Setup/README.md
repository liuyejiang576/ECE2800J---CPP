# README
This project allows you to set up your linux machine with some advanced features. First, you should run your linux machine and ***cd*** to the ***root*** directory. To achieve this, you need to run the following command in your terminal:

```bash
sudo -i
```

Now, you are under the ***root*** directory. Copy this five files (***init_env.sh***, ***install_vim.sh***, ***install_zsh.sh***, ***isoftstone_env.sh*** and ***Run.sh***) to this directory. You should not change the names of these files. Next, run the following command in your terminal:

```bash
bash Run.sh
```

Then, following the prompt to set up your linux machine. In this process, you need to initialize your ***machine name*** and ***user name***.

Next time you want you log into this user, you need to run the following command in your terminal:

```bash
su - <the user name>
```
